Sample Obsidian Vault (pre-converted by md2kindle)
===================================================

These .html files were generated from sample .md notes containing
[[wikilinks]]. They are fully self-contained with e-ink CSS.

Use this folder as your vault_root when testing obsidian.koplugin.

Files:
  Home.html          — links to Notes/Idea.html and Notes/Projects/Plan.html
  Notes/Idea.html    — links back to ../Home.html and Projects/Plan.html
  Notes/Projects/Plan.html  — links back to ../../Home.html and ../Idea.html

To add more notes later: run md2kindle (see converter/README.md) and
point it at this folder as the output_dir.
